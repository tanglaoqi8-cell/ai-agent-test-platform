# 极简 PRD（v0.1）

## 项目背景
团队需要一个可本地部署的 AI Agent 测试评估平台，用于统一管理 Prompt 与测试用例，并支持批量执行与结果查看。

## 目标用户
- Prompt 工程师
- AI 应用开发工程师
- 测试与评估人员

## v0.1 功能范围
- Prompt 管理：支持新增与列表查看
- CSV 用例上传：支持基础字段导入
- 模型配置：支持新增与列表查看
- 测试执行：基于 Prompt + 用例 + 模型触发执行
- 执行结果查看：按执行任务查看结果明细

## 暂不做的功能
- 多租户与权限系统
- 复杂工作流编排
- 在线标注与人工复审流
- 分布式调度与高可用部署

## 验收标准
- 项目目录结构完整，满足 monorepo 约定
- 文档（README、PRD、API 占位）齐全
- 提供可读取的示例测试数据 CSV
- 不包含业务实现代码与依赖安装产物
